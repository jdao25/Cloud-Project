CloudStorage is an online file storage web service for storing and accessing data on AWS Simple Storage Service (S3).
The service combines the performance and scalability of AWS Simple Storage Service with advanced security and sharing
capabilities. In using this application, we will be hosting a web server on Amazon Elastic Compute Cloud (EC2).
I use Amazon Simple Storage Service (S3) to store users' files securely within S3 buckets. Users will be able
to upload, delete, and download files from AWS S3 using this elegent website. In making an account, users will
be given 8 GB of storages to try for free! So what are you waiting for? Try CloudStorage TODAY! Click Here to register!
